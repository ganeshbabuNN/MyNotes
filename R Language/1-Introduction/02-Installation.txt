For Setting up the R in Windows
===============================
1)first install R Binary from the offical website https://cran.r-project.org/
2)download R for windows --> it will take few minutes
3)click on the binary icon--> click on the next , next until the installation is completed
4)Now for IDE , use RStudio https://www.rstudio.com/products/rstudio/download/
5)download the Rstudio Desktop
6)click on the next next, until the installation is completed.
7)now open R studio and test.

For Setting up the R in Linux
=============================
same as above step use the linux binary for both R and R studio

For Setting up the R in Mac
===========================
same as above step use the Mac binary for both R and R studio

For setup in Jupyter using online version & offline version
===========================================================
--https://jupyter.org/ --> go to try in browser --> click on R .. it takes take to setup.

#for offline version
Step 1: Open the Anaconda Prompt
Step 2: Add R to Jupyter
Next, type/copy the following command in order to add R to Jupyter:
conda install -c r r-irkernel
Next, type ‘y‘ and then press ENTER to proceed:
Step 3: Launch Jupyter Notebook
To launch Jupyter Notebook, first open the Anaconda Navigator. Then, click on the button to launch Jupyter.
Step 4: Create a new Notebook
To create a new Notebook for R:
First, click on ‘New‘ on the top right-hand-side of your screen
Finally, select ‘R‘ from the drop-down list
Step 5: Run your Code
You can now run your desired R code.
