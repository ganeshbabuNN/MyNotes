-As with other programming languages, You may like to store information of various data types like character, wide character, integer, floating point, double floating point, Boolean etc. Based on the data type of a variable, the operating system allocates memory and decides what can be stored in the reserved memory
-The variables are assigned with R-Objects and the data type of the R-object becomes the data type of the variable
-There are 7 data type
1)logical-TRUE,FALSE
2)numeric-12.3,5,999
3)integer-2L,34L,0L
4)complex-3+2i
5)character-"a", "Good"
6)raw
7)factor
-All these are detailed disussed in the further chapter
-To detect the type of data , use the typeof(object name) functions
-To get the size of the object , use object.size(object) functions.
