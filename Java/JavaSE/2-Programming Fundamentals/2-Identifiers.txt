Identifiers
===========
The name in a program is called an Identifier. 
Identifier is a sequence of characters. 
Identifiers are the names of the variables, methods, classes, packages, interfaces, arrays, enumerations, labeled names, etc

Example:
class Welcome{
public static void main(String[] args){ 
	int age  = 10; 
	System.out.println(?Welcome program?);
	}
}
