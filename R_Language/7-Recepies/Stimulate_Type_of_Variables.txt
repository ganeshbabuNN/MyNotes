### 1. **Use S3/S4 Classes**
- Define a class that wraps a value and enforces type checks.
- Example with S3:

```r
make_integer <- function(x) {
  if (!is.integer(x)) stop("Value must be integer")
  structure(x, class = "fixed_integer")
}

# Usage
a <- make_integer(5L)   # works
b <- make_integer(5)    # error: not integer
```

This way, you can only assign values of the intended type.

---

### 2. **Use R6 Classes**
R6 allows encapsulation and methods to enforce type constraints.

```r
library(R6)

FixedNumeric <- R6Class("FixedNumeric",
  private = list(value = NULL),
  public = list(
    initialize = function(x) {
      if (!is.numeric(x)) stop("Must be numeric")
      private$value <- x
    },
    set = function(x) {
      if (!is.numeric(x)) stop("Must be numeric")
      private$value <- x
    },
    get = function() private$value
  )
)

# Usage
num <- FixedNumeric$new(3.14)
num$get()
num$set(42)   # works
num$set("hi") # error
```

---

### 3. **Validation Wrappers**
Use functions like `stopifnot()` or `assertthat::assert_that()` to enforce type checks whenever you assign.

```r
set_fixed <- function(x, type = "numeric") {
  if (typeof(x) != type) stop("Wrong type")
  x
}

val <- set_fixed(10, "double")   # works
val <- set_fixed("hello", "double") # error
```

---

### 4. **Typed Environments**
Create an environment that enforces type rules when assigning.

```r
typed_env <- new.env()

assign_fixed <- function(name, value, type, env = typed_env) {
  if (typeof(value) != type) stop("Wrong type")
  assign(name, value, envir = env)
}

assign_fixed("x", 10, "double")
typed_env$x   # 10
assign_fixed("x", "hello", "double") # error
```
